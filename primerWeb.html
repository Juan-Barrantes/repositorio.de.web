Este es un parrafo simplecito simplecin xd

Pasos para hacer la vaina de git y git hub xd

1) hacer un git init en la carpeta donde está tu trabajo ese

2) vincula tu git con tu github: a. git config --global user.name "pones tu nombre" b. git config --global user.email "pones tu correo.com" c. Para verificar puedes hacer git --config --list

dentro de tu carpeta a tus archivos les haces las ediciones respectivas y en el VScode en el ícono de los puntos con las lineas notarás que te dirá cuantos cambios has hecho; debajo de cambios o changes te dirá qué arhivos han sido modificados y al costado derecho de estos mismos verás una serie de iconos, como el simbolo mas y otros, elige el simbolo mas y luego arriba en la barra de input de texto debes relatar qué cambios hubo, breve pero que sea entendible para el lector

Ahora veremos cómo subir todo lo que hicimos a Github

para eso crearemos una llave ssh, la configuramos y la subimos

creando la ssh:

ssh-keygen -t rsa -b 4096 -C "una palabra unica (puede ser tu correo)"

** Observaciones: luego del -b está el 4096 porq es el peso que le asignamos

damos enter enter hasta q salga la llave que es un gráfico medio raro xd

para ver donde está la llave hacemos: cd ~/.ssh y entre y luego una vez estando en esa dirección le damos ls para ver

OBS: el primer id que sale es el local, es super no lo compartas por NADA, y el otro que termina en punto pub (.pub) es tu llave pública

debemos verificar q tengamos nuestro servidor ssh correindo

eval "$(ssh-agent -s)"

si dice: Agent pid ###(algun numero) ; quiere decir que tu servidor de llave ssh está corriendo :3

ahroa agregamos la llave privada a este servidor

ssh-add ~/.ssh/nuestra llave privada

agregamos nuestra llave publica en github

entramos a github, vamos a 'Settings', luego buscamos 'SSH and GPG keys', ahora vamos dentro de la terminal y para agregar la llave publica antes ingresamos: sudo apt install xclip (nos permitirá hacerlo de forma segura)

luego hacemos en la terminal --> xclip -selection clipboard <~/.ssh/id_rsa.pub ; es para copiar en el portapapeles la llave publica de manera segura

luego me voy al gitHub, le doy en nueba llave (new key), ingreso un nombre para nuestra llavecita :3 y en donde dice 'Add SSH key' ingresamo nuestra llave publica que ya tenemos en el portapapeles

ahora subiremos al respositorio lo q hemos hecho ._.
